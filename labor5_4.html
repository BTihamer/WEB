Web‑CV feladatsor (külön HTML fájlban dolgozol)

Készíts saját index.html fájlt, és add hozzá a profilkártyát. Az alábbi lépéseket követve a végén egy kész, sötét témás CV oldalad lesz.

Feladat 1 — Saját HTML létrehozása

  1. Hozz létre egy portfolio.html fájlt a gépeden.
  2. Adj hozzá alap HTML vázat (<!DOCTYPE html>, <html>, <head>, <body>), és állíts be sötét témát (háttér, szövegszín).
  3. Linkeld be saját style.css fájlod (vagy használj <style>‑t a head‑ben).
Kezdő sablon (másolható) 
<!DOCTYPE html>
<html lang="hu">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Portfólió / CV</title>
  <meta name="color-scheme" content="dark">
  <style>
    body{background:#0b0f14;color:#e6e9ef;font-family:system-ui, -apple-system, Segoe UI, Roboto, Arial;margin:0;padding:24px}
  </style>
</head>
<body>
  <main id="app"></main>
</body>
</html>

Feladat 2 — Profilkártya hozzáadása a saját oldaladhoz

Készíts egy kártyát profilfotóval, névvel, pozícióval és linkekkel (email, LinkedIn, GitHub). Ez a kártya később a CV tetején fog megjelenni.

Illeszd a következőt a <main id="app"> belsejébe:

<section class="profile-card">
  <img class="avatar" src="te.jpg" alt="Profilkép" />
  <div class="meta">
    <h1 class="name">Név Vezetéknév</h1>
    <p class="role">Pozíció · Szakterület</p>
    <div class="links">
      <a href="mailto:te@pelda.hu">Email</a> ·
      <a href="https://linkedin.com/in/te">LinkedIn</a> ·
      <a href="https://github.com/te">GitHub</a>
    </div>
  </div>
</section>

Add hozzá ezt a minimál CSS‑t a kártyához (head <style>‑ba mehet):

.profile-card{display:flex;gap:18px;align-items:center;background:#11161d;border:1px solid #1f2937;border-radius:16px;padding:16px;box-shadow:0 10px 30px rgba(0,0,0,.35)}
.profile-card .avatar{width:96px;height:96px;border-radius:50%;object-fit:cover;border:2px solid #1f2937}
.profile-card .name{margin:0;font-size:28px}
.profile-card .role{margin:4px 0 8px;color:#9aa4b2}
.profile-card .links a{color:#e6e9ef;text-decoration:none}
.profile-card .links a:hover{text-decoration:underline}

✅ Elfogadási kritériumok:

  • A kártya reszponzív, 600px alatt a kép a név fölé kerül (media query).
  • Kontrasztos, sötét háttér és világos szöveg.
  • Képfájl (vagy placeholder URL) működik.
Reszponzív töréspont példa 
@media (max-width: 600px){
  .profile-card{flex-direction:column;align-items:flex-start}
  .profile-card .avatar{width:84px;height:84px}
}

Tesztkép hiányában használhatsz placeholdert: https://picsum.photos/200 vagy https://i.pravatar.cc/200.

Feladat 3 — CV szekciók (a saját fájlodban)

  1. Hozz létre szekciókat: Bemutatkozás, Tapasztalat, Oktatás, Készségek, Kapcsolat.
  2. Mindegyik szekció legyen kártya (sötét panel, kerekített sarok, belső padding).
  3. Tapasztalatnál legalább 2 munkahely, mindegyiknél 2–3 bullet.
  4. Készségeknél “chip”‑szerű címkék (pl. Python, React...).
Chip példa 
.chip{display:inline-block;margin:4px 6px 0 0;padding:6px 10px;border:1px solid #1f2937;border-radius:999px;background:rgba(255,255,255,.03);font-size:12px}

Feladat 4 — Linkek és SEO apróságok

  1. Adj meg címet a lapnak (pl. “Név – Portfólió / CV”).
  2. Töltsd ki a <meta name="description">‑t rövid összegzéssel.
  3. Minden külső link új lapon nyíljon: target="_blank" rel="noopener".
  4. Favicon opcionális.

Feladat 5 — Publikálás

  1. Nyisd meg a portfolio.html‑t a böngészőben és ellenőrizd a reszponzivitást.
  2. Töltsd fel GitHub Pagesra.
