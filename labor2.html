CSS Gyakorlatok

Haladj sorrendben – a feladatok egymásra épülnek.

1. Külső stíluslap csatolása

Hozz létre egy style.css fájlt, és csatold a <head>-ben.

Tipp: <link rel="stylesheet" href="style.css">

2. Alap reset és dobozmodell

Állíts be alapértelmezéseket a konzisztens megjelenéshez.

* { box-sizing: border-box; margin: 0; padding: 0; }
html, body { height: 100%; }

3. Színek és tipográfia

Színezd meg a címsorokat, állíts be alap betűtípust és -méretet.

h1 { color: royalblue; }
h2 { color: seagreen; }
body { font-family: Arial, sans-serif; font-size: 16px; }

4. Térközök és bekezdések

Adj külső margót a bekezdéseknek és kényelmes sortávolságot.

p { margin: 20px 0; line-height: 1.7; }

5. Háttérszín és kontraszt

Adj világos háttérszínt az oldalhoz, figyelve az olvashatóságra.

body { background-color: #f4f8ff; color: #222; }

6. Gomb-stílus és hover állapot

Stilizálj egy gombot és adj rá lebegéskor színváltást.

button {
  background-color: orange; color: white; border: none;
  padding: 10px 16px; border-radius: 6px; cursor: pointer;
}
button:hover { background-color: darkorange; }

7. Állapotok és fókusz-jelzés

Biztosíts látható fókuszt billentyűzetnél is.

button:focus, a:focus, input:focus, textarea:focus {
  outline: 3px solid cornflowerblue; outline-offset: 2px;
}

8. Képek formázása

Kerekített sarkok vagy kör alakú avatar kép.

img { max-width: 100%; height: auto; border-radius: 12px; }
.avatar { width: 160px; height: 160px; border-radius: 50%; object-fit: cover; }

9. Listák testreszabása

Rendezett listát római számokkal és extra beljebb húzással.

ol { list-style-type: upper-roman; margin-left: 30px; }

10. Flexbox kétoszlopos elrendezés

Hozz létre rugalmas, kétoszlopos elrendezést.

.container { display: flex; gap: 20px; }
.container > div { flex: 1; padding: 10px; background: #fff; border-radius: 8px; }

11. Grid elrendezés és reszponzivitás

Alakíts galériát CSS Griddel, törésponttal telefonra.

.grid {
  display: grid; gap: 16px;
  grid-template-columns: repeat(3, 1fr);
}
@media (max-width: 800px) {
  .grid { grid-template-columns: repeat(2, 1fr); }
}
@media (max-width: 520px) {
  .grid { grid-template-columns: 1fr; }
}

12. Betűk a Google Fonts-ból

Importálj egy betűtípust (pl. Roboto) és alkalmazd az oldalon.

@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');
body { font-family: 'Roboto', system-ui, -apple-system, sans-serif; }

13. Inline, belső és külső CSS gyakorlat

  1. Készíts 2 példát inline CSS-re (pl. egy-egy színezett szó).
  2. Készíts 2 példát belső (internal) CSS-re a <style>-ban.
  3. Készíts 2 külön külső CSS fájlt (pl. base.css és components.css) és csatold mindkettőt.
<link rel="stylesheet" href="base.css">
<link rel="stylesheet" href="components.css">

14. Finomítás: átmenetek és animáció alapok

Adj sima átmenetet a linkek és a gombok lebegéséhez.

a, button { transition: background-color 0.2s, color 0.2s, transform 0.2s; }
button:hover { transform: translateY(-1px); }

15. Összerendezés és esztétika

Dolgozz önállóan: tedd a weboldalt letisztulttá és érthetővé. Figyelj a konzisztenciára (távolságok, színek, tipográfia, fókusz).

  • Egységes térközskála (pl. 8px-es lépések).
  • Egységes gomb- és linkstílus.
  • Reszponzív ellenőrzés 360–1440px között.
