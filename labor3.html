JavaScript Gyakorlatok

A feladatok egymásra épülnek: kezdj az alapokkal, majd haladj a DOM és események felé!

1. Két szám összeadása

Készíts egy függvényt, amely két számot paraméterként vesz át és visszaadja az összegüket.

function add(a, b) {
  return a + b;
}
console.log(add(3, 4)); // 7

2. Páros vagy páratlan?

Írj függvényt, amely visszaadja, hogy egy szám páros-e vagy páratlan.

function isEven(num) {
  return num % 2 === 0;
}
console.log(isEven(4)); // true
console.log(isEven(5)); // false

3. Számláló gombnyomásra

Készíts egy számlálót, amely növeli az értékét gombnyomáskor.

<button onclick="incrementCounter()">Növelés</button>
<p id="counter">0</p>

<script>
let count = 0;
function incrementCounter() {
  count++;
  document.getElementById('counter').innerText = count;
}
</script>

4. Adat bekérése a felhasználótól

Kérj be egy nevet prompt()-tal, majd írd ki az oldalon.

<button onclick="askName()">Kérdezd meg a nevét</button>
<p id="name"></p>

<script>
function askName() {
  let name = prompt('Mi a neved?');
  document.getElementById('name').innerText = 'Üdvözöllek, ' + name + '!';
}
</script>

5. Visszaszámláló

Készíts egy 10 másodpercről induló visszaszámlálót.

<button onclick="startCountdown()">Indítsd el</button>
<p id="countdown">10</p>

<script>
function startCountdown() {
  let timeLeft = 10;
  let timer = setInterval(function() {
    if (timeLeft <= 0) {
      clearInterval(timer);
      document.getElementById('countdown').innerText = 'Idő lejárt!';
    } else {
      document.getElementById('countdown').innerText = timeLeft;
    }
    timeLeft--;
  }, 1000);
}
</script>

6. Tömb és lista

Hozz létre egy tömböt, majd jelenítsd meg az elemeit a weboldalon.

<ul id="fruitList"></ul>

<script>
let fruits = ['Alma', 'Banán', 'Narancs'];
fruits.forEach(function(fruit) {
  document.getElementById('fruitList').innerHTML += '<li>' + fruit + '</li>';
});
</script>

7. Tömb elemeinek szorzása

Írj függvényt, amely minden tömbelemre szorzót alkalmaz.

function multiplyArray(arr, multiplier) {
  return arr.map(num => num * multiplier);
}
let numbers = [1, 2, 3, 4];
console.log(multiplyArray(numbers, 2)); // [2,4,6,8]

8. Objektum tulajdonságok megjelenítése

Jelenítsd meg egy objektum adatait listában.

<ul id="personInfo"></ul>

<script>
let person = { name: 'John', age: 30, city: 'Budapest' };
for (let key in person) {
  document.getElementById('personInfo').innerHTML +=
    '<li>' + key + ': ' + person[key] + '</li>';
}
</script>

9. Eseménykezelő kattintásra

Írj egy gombot, amely kattintáskor megváltoztatja a háttérszínt.

<button onclick="changeBg()">Háttér váltás</button>

<script>
function changeBg() {
  document.body.style.backgroundColor = 'lightcoral';
}
</script>

10. Időzítő üdvözlő üzenet

Használj setTimeout()-ot, hogy 5 másodperc után üzenetet írj ki.

<script>
window.onload = function() {
  setTimeout(function() {
    alert('Üdvözöllek az oldalon!');
  }, 5000);
}
</script>
